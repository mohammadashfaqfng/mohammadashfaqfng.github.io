# Mohammad Ashfaq — Portfolio Website

## Files
- `index.html` — Main portfolio page
- `style.css` — Stylesheet (dark industrial theme)
- `script.js` — Animations and interactions
- `Mohammad_Ashfaq_CV.pdf` — Your CV (you must add this file)

## How to Publish on GitHub Pages

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
- Click the "+" icon → "New repository"
- Name it exactly: `your-username.github.io`
  (replace "your-username" with your actual GitHub username)
- Set it to Public
- Click "Create repository"

### Step 3 — Upload files
- Click "uploading an existing file"
- Upload all 4 files:
  - index.html
  - style.css
  - script.js
  - Mohammad_Ashfaq_CV.pdf
- Click "Commit changes"

### Step 4 — Enable GitHub Pages
- Go to your repository Settings
- Click "Pages" in the left sidebar
- Under "Source" select "Deploy from a branch"
- Select "main" branch → "/ (root)"
- Click Save

### Step 5 — Your site is live!
Visit: https://your-username.github.io

It may take 2–5 minutes to go live the first time.

## Vibe Coding (Editing) Tips

To edit content, open `index.html` in any text editor (Notepad, VS Code, etc.)

- **Change your name/title**: Search for "Mohammad Ashfaq" and update
- **Update contact info**: Search for "mohdashfaq2003" 
- **Add/remove experience**: Copy a `<div class="tl-item">` block and edit
- **Add company logos**: Replace the `src=` URL in `<img>` tags with a direct image URL
- **Change accent color**: In style.css, change `--amber: #e07b2a;` to any color

After editing, re-upload the changed files to GitHub and changes go live in ~1 min.
